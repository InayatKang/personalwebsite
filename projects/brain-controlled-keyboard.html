Brain Controlled Keyboard

Detecting blinks via EEG to control keyboard commands.

Overview

Using the Muse headset, a meditation band, I reverse engineered it, extracting my brainwaves and using them to identify changes in delta wave values to detect blinks and control keyboard commands.

This empathy-oriented design focuses on improving accessibility for those with limited mobility, such as ALS patients.

Problem

People with limited mobility, including ALS patients, often lack accessible ways to interact with computers using conventional input devices. Traditional assistive tech can be expensive or poorly matched to real user constraints.

Approach

I used a Muse meditation headset to stream EEG data, identified blink-related changes in delta wave values, and mapped those signals to keyboard commands, creating a low-cost, brain-controlled input pathway.

Future directions

Extend command vocabulary, improve signal robustness in real environments, and explore clinical partnerships for assistive use cases.
